# James Vamos Portfolio

Public-safe portfolio for selected professional work, technical writing, and operations-focused case studies.

This repository is meant to be shareable with:
- teachers
- peers
- collaborators
- hiring managers
- recruiters

It is not a raw work archive.
It is a curated showcase built from real experience and rewritten for safe public viewing.

## What This Portfolio Emphasizes

My strongest work tends to sit at the intersection of:
- operations and reliability
- technical documentation
- implementation support
- risk and controls thinking
- process improvement
- translating complex technical work into clear execution

## Public-Safe Sharing Standard

Everything here is intentionally sanitized.
I do not publish employer-confidential information, raw internal documents, customer-sensitive details, credentials, private infrastructure details, or proprietary materials.

If a project cannot be shared directly, I rewrite it as a case study that preserves the technical truth while removing sensitive context.

See [PUBLIC_PORTFOLIO_POLICY.md](./PUBLIC_PORTFOLIO_POLICY.md).

## Portfolio Principles

- Ground everything in real work.
- Keep chronology and responsibility accurate.
- Prefer clear outcomes over inflated language.
- Use public-safe summaries instead of raw internal artifacts.
- Avoid duplicate examples when one stronger example says it better.

## Current Scope

This public repo is the showcase layer.
The larger career archive and intake pipeline remain private.
Only selected, rewritten, shareable material should live here.
